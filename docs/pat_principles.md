# PAT-Style Principles Used Here

This harness follows four operational principles:

1. Segment the document into logical components.
2. Allocate more budget to dense or high-risk sections.
3. Run specialised reviewers with the full manuscript available as context.
4. Synthesize, deduplicate, ground, and adjudicate before patching.

The important adaptation for mathematics papers is that adjudication is mandatory. A pile of model critiques is not a review; it becomes useful only after the issue ledger has been filtered by strict acceptance rules.
