# Workflow

The harness deliberately separates discovery, judgement, patching, and regression.

```text
0. Bootstrap scaffold
1. Repository inspection
2. Build and source hygiene
3. Segment and budget
4. Theorem/assumption/equation inventory
5. Specialised audits
6. Global synthesis into proposed issue ledger
7. Adjudication
8. Patch planning
9. Accepted patches only
10. Regression audit
11. Final gate
```

Do not skip adjudication. It is the precision-control mechanism.
