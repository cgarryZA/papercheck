# Mathematical Failure Mode Taxonomy

Use these categories when writing issues:

- Undefined object
- Used before defined
- Insufficient assumption
- Hidden regularity
- Hidden measurability/adaptedness/predictability
- Integrability gap
- Compactness gap
- Limiting argument gap
- Incorrect norm/topology
- Constant depends on forbidden quantity
- Circular dependency
- Cited theorem outside hypotheses
- Sign error
- Missing boundary/initial/terminal term
- Double-counted term
- Dropped error term
- Claim stronger than proof
- Numerical metric mismatch
- Reproducibility gap
- Novelty/related-work overclaim
- Source hygiene blocker
